- 항상 과제 제출 방식은 자신의 브랜치를 만들어서 해당 브랜치에만 올립니다. 개인 브랜치 명은  
 (ex. 과제번호-전공-이름) // 1-iOS-Seojiwan

- 과제를 완료하면 PR을 올리고 선배들의 코멘트를 확인한 후 2개 이상의 어프로브와 코멘트 답장 및 수정을 모두  
완료 했을때 자신의 전공에 맞는 브랜츠로 머지를 시킵니다. Front이면 FrontEnd 브랜치로 머지.
